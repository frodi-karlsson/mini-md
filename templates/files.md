# Hosting files
You can upload files to the `assets` directory and link to them in your markdown files. For example, you can upload an image to `assets/images` and link to it in your markdown file like this:
```markdown
![alt text](/assets/image-name.png)
```
![sample image](/assets/image.webp)

Or a different file type:
```markdown
[link text](/assets/file-name.pdf)
```
[Here's an example txt file!](/assets/textfile.txt)

[Here's a pdf!](/assets/helloworld.pdf)
