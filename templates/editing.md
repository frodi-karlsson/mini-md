# Editing

Although you should check out the [documentation](https://www.markdownguide.org/basic-syntax/) for markdown, here are some of the most common things you might want to do:

## Adding images

You can add images to your markdown files using the following syntax:

```markdown
![alt text](/assets/image-name.png)
```

## Adding links

You can add links to your markdown files using the following syntax:

```markdown
[link text](/path/to/page)
```
or for external links:
```markdown
[link text](https://example.com)
```

## Adding code blocks

You can add code blocks (the ones that contain all the examples in this documentation) to your markdown files using the following syntax:
````markdown
```python
print("Hello world!")
```
```markdown
This is how i made this page!
```
````

## Headers and text formatting

You can add headers and format text using the following syntax:

```markdown
# Header 1
## Header 2
### Header 3
These are some words in *italics* and some in **bold**.
```
# Header 1
## Header 2
### Header 3
These are some words in *italics* and some in **bold**.

## Lists

You can add lists using the following syntax:

```markdown
- Item 1
- Item 2
- Item 3
```
or checklists:
```markdown
-[x] Item 1

-[ ] Item 2

-[ ] Item 3
```
- Item 1
- Item 2
- Item 3

and checklists:

-[x] Item 1

-[ ] Item 2

-[ ] Item 3
