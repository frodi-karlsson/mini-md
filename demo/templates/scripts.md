# Adding Scripts

Adding a script to a template is as easy as creating a scripts in your script directory that starts with the name of the template. For example, if you have a template called `home`, you can add a script called `home-script.js` to your script directory. This script will be loaded when the `home` template is loaded.

In this instance we have a script called scripts-hello.js that logs hello to the console. This script is loaded when the `scripts` template is loaded.
